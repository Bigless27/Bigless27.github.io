What are the differences between relative, absolute, fixed, and static positioning? This is a great question that took me a long time to figure out the answers too. When I first learned the concepts none of the information stuck. I was constantly looking back over material to help solve this problem. Finally after many time constantly reviewing the material I can finally say I what these forms of positioning do. Relative positioning positions the element relative to other elements on the page. This is the elements normal position and will move over for other elements to create space. Absolute positioning on the other hand doesn’t care about other elements on the page it places itself in relation to its parent element. If no parent element is stated in the html then the absolute positioned element parent is the entire web page itself. This is good for placing an image exactly where you want on the page.

Static positioning is html default placement of elements. Static elements don’t have similar properties of relative and absolute like top, bottom, left and right. These elements follow the flow of your html code. Finally, Fixed positioning is very interesting. Any element placed in a fixed position is relative to the users viewpoint. This means wherever you place an element in a fixed position it will stay in that spot, even if the user scrolls down. This make it easy for things like navigation bars to always be accessible so the user doesn’t have to scroll all the way up to access content. I personally prefer using fixed positioning for elements like logos. The benefit of this is that if a person was to randomly jump to any page of you website they’ll know that our company is behind the content.

Something that I forgot to mention about absolute positioning is that you need to type in the dimension for where to place the element. These are the top, right, left, and bottom properties. When typing in the values for these properties it is good to mention that you should use the “em” unit instead of “px”, this is because of the way computer screens display pixels. Some screens may be smaller than others and lead to distortion of the placement of your content. The “em” unit adjusts your dimensions relative to the users screen so you don’t have to worry the positioning of your elects on different screens.
